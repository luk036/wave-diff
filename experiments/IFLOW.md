# IFLOW.md - 波形差异分析实验项目上下文

## 项目概述

当前目录 `D:\github\py\wave-diff\experiments` 包含一系列用于波形和文本差异分析的 Python 实验工具。这些工具专注于使用编辑距离算法（如 Levenshtein 距离）和动态时间规整（DTW）技术来比较和分析不同类型的数据序列。

## 项目文件结构

### 核心文件

1. **waveform_diff.py** - 主要的波形比较工具
   - **功能**: 使用编辑距离技术和动态时间规整比较两个波形
   - **特性**: 
     - 支持从文件加载波形或生成合成信号（正弦波、方波、三角波、啁啾信号、ECG模拟信号）
     - 实现动态时间规整（DTW）算法用于波形对齐
     - 提供可视化比较结果，包括原始波形、对齐后波形和操作统计
     - 生成详细的文本报告
   - **依赖**: numpy, matplotlib, scipy (可选)
   - **用法示例**:
     ```bash
     # 比较两个信号文件
     python waveform_diff.py signal1.txt signal2.txt
     
     # 比较合成信号
     python waveform_diff.py sine square --synthetic
     
     # 自定义比较参数
     python waveform_diff.py file1.txt file2.txt --threshold 0.05 --window 100 --no-warping
     ```

2. **diff_distance.py** - 文本编辑距离计算工具
   - **功能**: 计算两个文本文件之间的 Levenshtein 编辑距离
   - **特性**:
     - 字符级、单词级和行级编辑距离计算
     - 详细的编辑操作路径回溯
     - 相似度百分比计算
     - 可视化动态规划矩阵
   - **用法示例**:
     ```bash
     # 基本比较
     python diff_distance.py file1.txt file2.txt
     
     # 显示详细编辑操作
     python diff_distance.py file1.txt file2.txt --detailed
     
     # 显示DP矩阵
     python diff_distance.py file1.txt file2.txt --matrix
     ```

3. **diff_tool.py** - 通用文本差异工具
   - **功能**: 提供多种格式的文本差异比较
   - **特性**:
     - 支持简单、统一、上下文和HTML格式的差异输出
     - 可配置上下文行数
     - 支持输出到文件
   - **用法示例**:
     ```bash
     # 统一格式差异（类似git diff）
     python diff_tool.py file1.txt file2.txt --format unified
     
     # HTML格式差异
     python diff_tool.py file1.txt file2.txt --format html -o diff.html
     ```

4. **waveform_diff_old.py** - 旧版波形比较工具
   - **功能**: 早期版本的波形比较实现
   - **特点**: 包含更多实验性功能，如时间偏移操作类型
   - **状态**: 已被 `waveform_diff.py` 取代，保留作为参考

## 技术架构

### 核心算法

1. **编辑距离算法**:
   - Levenshtein 距离用于文本比较
   - 修改版本用于波形幅度差异比较

2. **动态时间规整 (DTW)**:
   - 用于处理不同长度或时间偏移的波形对齐
   - 简化实现以避免内存问题

3. **滑动窗口比较**:
   - 用于波形的局部差异检测
   - 可配置窗口大小和阈值

### 数据处理流程

1. **波形处理**:
   ```
   输入文件/合成信号 → 归一化 → DTW对齐 → 编辑操作检测 → 可视化/报告
   ```

2. **文本处理**:
   ```
   文件读取 → 分割(字符/单词/行) → DP矩阵计算 → 编辑路径回溯 → 结果输出
   ```
